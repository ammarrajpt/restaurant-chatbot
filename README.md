### 1. 🍽️ Restaurant AI Chatbot (`restaurant_chatbot.py`)

An intelligent chatbot for a restaurant that answers customer queries about menu, prices, and orders.

**Features:**

- View full menu with prices
- Ask price of any item
- Place orders
- Get restaurant timing and location info
- Understands Urdu words (kitna, chahiye, kahan)

**Technologies Used:**

- Python
- NLTK (Natural Language Processing)
- Tokenization & Pattern Matching

**How to Run:**

```bash
pip install nltk
python restaurant_chatbot.py
```

### 2. 📷 Face Detection Attendance System (`face_attendance.py`)

A real-time face detection system that automatically marks and records student attendance.

**Features:**

- Real-time face detection using webcam
- Automatic attendance marking
- CSV file generation with date & time
- Duplicate prevention (marks once per day)
- Color coding (Green = new, Blue = already marked)

**Technologies Used:**

- Python
- OpenCV
- Haar Cascade Classifier
- CSV (Data Storage)

**How to Run:**

```bash
pip install opencv-python
python face_attendance.py
```
